### Fork info
This is a heavily modified, stripped down copy for use with Loton, here: https://github.com/loganmcbroom/Loton

### Build
Have Qt installed
git clone https://github.com/loganmcbroom/nodeeditor
cd nodeeditor
cmake -S . -B build
cmake --build build --target install
